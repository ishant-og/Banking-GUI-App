# 🏦 Banking GUI Application

A simple and beginner-friendly Banking System GUI application built using **Python Tkinter**. This project allows users to create accounts, log in securely, deposit money, withdraw money, check balance, and view personal details through a graphical user interface.

---

# 🚀 Features

✅ User Registration System
✅ Secure Login Authentication
✅ Deposit Money
✅ Withdraw Money
✅ Balance Checking
✅ Personal Details Dashboard
✅ GUI Interface using Tkinter
✅ File Handling for Data Storage
✅ Basic Banking Workflow Simulation

---

# ⚙️ How It Works

## 1️⃣ Registration

Users can create a new bank account by entering:

* Name
* Age
* Gender
* Password
* Contact Number

The data is stored locally using text files.

---

## 2️⃣ Login

Users can log in using:

* Username
* Password

After successful login, the dashboard opens.

---

## 3️⃣ Dashboard Features

After login, users can:

* View Personal Details
* Deposit Money
* Withdraw Money
* Check Account Balance

---

# 💰 Deposit Functionality

Users can deposit money into their account.

Validation included:

* Empty amount check
* Negative amount restriction

---

# 💸 Withdraw Functionality

Users can withdraw money from their account.

Validation included:

* Empty amount check
* Negative amount restriction
* Insufficient balance check
