# Banking GUI App

A modern Banking GUI Application developed to simulate core banking operations with an interactive graphical user interface.

## Features

* User-friendly banking dashboard
* Deposit and withdrawal operations
* Balance checking
* Account management
* GUI-based interaction
* Secure and organized workflow

## Tech Stack

* Java
* Java Swing / AWT
* OOPs Concepts
